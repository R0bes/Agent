
# configforge

**ConfigForge** is a Python utility for handling application configuration in a standardized,
production-ready way. It supports:

- **Pydantic** schema definitions for config validation
- **TOML** config file generation & loading
- **.env** variable parsing
- **argparse** command-line argument helpers
- **on_config_change** callbacks

## Installation
```bash
pip install pydantic toml
```
